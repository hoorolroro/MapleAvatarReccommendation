## 개발 과정
 - 크롤링을 바탕으로 메이플 코디와 아이템을 수집한다.
 
 - CNN을 바탕으로 코디와 아이템을 색상별로 학습한다.
 
 - GUI 기반 실행파일을 만들어 사용자에게 배포한다.
  
 - 사용자의 피드백을 데이터베이스에 쌓아서 더 나은 코디를 추천한다.
